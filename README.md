# Mapa da Operação - Comunikime

A B2B consultive web micro-application designed for pre-diagnostic operations for Comunikime's potential clients.

## Features (Phase 1)
- Core data models for B2B client diagnostic collection.
- Internal calculation of lead score, priority (A, B, C, D) and operational heat/temperature.
- Solution mapping matching operational profiles and pains to Comunikime's platforms.
- Automatically generated operational summary and pending questions for SDR review.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
